## üéì Best Practice Insights (2025)

### React/Next.js Architecture
- **Server Components by default** (minimize client JavaScript)
- **Modular folder structure** (/features or /domains pattern)
- **Minimal state libraries** (prefer Context API over Redux)
- **Core Web Vitals SLOs** (LCP, INP, CLS targets)
- **Accessibility first** (WCAG 2.2 AA compliance)

**Sources:** Strapi, RaftLabs, SoftwareMill, Robin Wieruch (2025 articles)

### LLM Orchestration Frameworks
- **ReAct (Reason + Act):** Best for FlashFusion generation workflow
- **Chain-of-Thought:** For explaining complex decisions to users
- **Tree-of-Thoughts:** Optional for platform selection branching
- **Automatic Prompt Engineering:** Future optimization opportunity

**Sources:** Prompt Engineering Guide, Width.ai, Mercity.ai
